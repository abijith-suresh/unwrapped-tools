# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

## [0.4.0] - 2026-04-05

### Added

- Stable post-rebrand polish for unwrapped.tools with informational pages, settings polish, and a clearer multi-tool shell.

## [0.3.0] - 2026-04-01

### Added

- The first complete unwrapped.tools multi-tool platform after the twish rebrand, including the IDE-style shell and full tool suite.

## [0.2.0] - 2026-03-28

### Added

- The strongest pre-rebrand twish milestone with the config-compare workflow and an explicit major-quality product release candidate.

## [0.1.0] - 2026-02-22

### Added

- Initial twish public release with landing/docs pages, diff tooling, PWA support, and open-source setup.
